## Introduction
Dans ce laboratoire, vous allez expérimenter la technologie Voice-to-Text en utilisant le modèle Whisper d'OpenAI.

## Objectifs
L'objectif est de comprendre  comment transcrire de l'audio en texte à l'aide d'un modèle d'intelligence artificielle. Pour simplifier l'installation, nous utiliserons Docker. Donc vous devez l'installer sur votre ordinateur. Le labo va être codé en Python.

## Avant de commencer !
Clonez ce dépôt Git : https://github.com/edwin2346/atelier-whisper.git dans un dossier que vous allez ouvrir sur Visual Studio Code.

Il y a déjà un audio dans le dépôt Git, mais si vous voulez, vous pouvez mettre le vôtre à l'aide de l'enregistreur vocal de Windows.

Vous n'aurez pas à écrire de code, vous allez juste devoir copier-coller le code que je vous donne et le tester.


### Étape 1 : Ouvrir le projet
1. Ouvrez Visual Studio Code
2. Ouvrez le dossier que vous avez cloné

### Étape 2 : Démarrer Docker
Ouvrez le terminal dans VS Code et tapez cette commande :
```bash
docker-compose up
```
Attendez que tout s'installe (quelques minutes).

### Étape 3 : Suivre les instructions 
Après avoir fait ces étapes, il vous reste juste à suivre les instructions dans les autres pages du labo.
